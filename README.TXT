------------------------------------------------------------------------
REAL ESTATE PROJECT
------------------------------------------------------------------------

PROJECT TITLE: RealEstate
PURPOSE OF PROJECT: Simulate a Real-Estate Desktop Application
VERSION or DATE: Final Version 4.0
HOW TO START THIS PROJECT: Type in any house-range $100,000 and up
AUTHORS: April McBroom
USER INSTRUCTIONS: Upon launching, type in the amount for house. The number of bathrooms, rooms, boolean yard affects search. To add a house, type in the full information
in each value slot including text-area for description, then press Add afterwards. All custom pictures must be in the same numbering order pattern and name as the housephotos in order to be valid.
The house numbering window is for debugging purposes and can be turned off via clicking the x button on the upper-right hand corner.
